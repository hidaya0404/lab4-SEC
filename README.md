# Analyse Statique Android - UnCrackable Level 1

## Présentation du projet

Ce projet présente une analyse statique d’une application Android pédagogique : **UnCrackable Level 1** de l’OWASP MSTG.

L’objectif principal est de comprendre la structure interne d’un APK Android, d’analyser son manifeste, d’identifier les composants déclarés, de rechercher des chaînes sensibles et de comparer différents outils de décompilation Android.

Cette analyse s’inscrit dans un cadre strictement pédagogique et légal.
